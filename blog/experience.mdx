# Exprience

### GaeShow

개쇼는 개발자가 쓰는 아이템의 정보를 공유하고 팁을 나눌수있는 커뮤니티를 만들자 라는 아이디어에서 출발한 프로젝트입니다.   
저는 이 프로젝트에서 ios 개발을 맡아서 하였습니다.
  
  
- 기여도 : 100%
- 개발기간 : 18주
- 개발언어 : swift
- 주요 개발 기능 
  - 회원가입/ 로그인 (카카오 로그인 구현)
  - 게시판, 댓글 기능
  - 차트를 활용한 그래프 보기 페이지 구현
  - 검색기능



- 프로젝트 진행시 생겼던 문제 및 해결방안  
  ❓  swift ui를 지원하지 않는 외부 라이브러리를 사용해야하는 문제 (ex. rich editor, google,kakao login)  
  ✅  최대한 swift 최신버전을 지원하는 라이브러리를 찾은다음에 UIViewRepresentable을 이용하여서 swift ui에 적용시켰습니다. 
  
  ❓  main tabview를 눌렀을때 모든 뷰에서 리로드가 되어서 서버요청을 계속 보내는 현상   
  ✅  swiftui의 main tabview를 사용한것을 UIHostingController를 만들어서 uikit tabview 구조체를 만들어서 사용하였습니다.  
  연속적인 요청을 막을수 있었습니다.

### 랜선 집들이

랜선집들이는 우리집에 가구를 배치할때 미리보고싶은 유저를 위해서 만든 앱입니다.  
유저의 집에 ar로 가구를 배치하여 우리집에 어울리는 가구를 볼수있는 앱입니다.
  
  
- 기여도 : 100%
- 개발기간 : 16주
- 개발언어 : Java
- 주요 개발 기능 
  - arcore 사용하여 가구배치
  - 이미지에 태그걸기 기능
  - 카카오로그인 구현
  - 부트페이 api를 이용해서 결제 가능


- 프로젝트 진행시 생겼던 문제 및 해결방안  
  ❓  arcore를 사용하여 가구 배치를 할때 여러개의 가구를 띄웠을때 그 가구들을 하나하나 지울때 전체가 지워졌던 문제  
  ✅  공식문서를 찾아보니 화면에 생성되는 ar 오브젝트는 node라는 오브젝트인데 이 node를 터치하면 선택할수있는 핸들러 메소드를 찾아서 원하는 node를 클릭하고 그 node의 Anchor값에 null을 넣어주어서 해결하였다.
  
  ❓  가상으로 배치한 모델의 색상을 바꾸고자 했을때 내가 원하는곳에 모델만 색상이 바뀌어야 하는데 모델 전체의 색상이 바뀌었던 문제  
  ✅  Seanfrom에서 .sfb파일을 만들때 어떠한 부분이 색상을 담당하는 부분인지 찾아보았고 그 단어와 검색어를 조합하여서 seanform에서 제공하는 함수 중 해당 부분의 색상을 바꾸는 함수(submashindex)를 하나하나 대조하고 숫자를 바꿔보면서 부분적으로 색상을 바꾸어주는 함수를 찾아서 적용시켜주었다.
